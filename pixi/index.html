    Install with Pixi

    Pixi is a fast, reproducible environment manager. See the official docs: https://pixi.sh/latest/

    This repository includes ready-to-use Pixi environments:

    • pixi/microsam_cellpose3: micro_sam + cellpose + MONAI (+ PyTorch GPU on Linux/Windows when available; CPU-only on macOS arm64)
    • pixi/stardist: StarDist + TensorFlow + Napari

    Quick start

    1) Install Pixi (follow the instructions at https://pixi.sh/latest/). 2) Choose an environment folder and run:

    cd pixi/microsam_cellpose3   # or: cd pixi/stardist
pixi install                 # resolves and creates the environment
pixi run startup             # launches the app in the configured environment

    Notes

    • You can alternatively drop into the environment with pixi shell and run python startup.py.
    • GPU/CPU targets are encoded in each pixi.toml; ensure your system drivers/toolkit match if you expect GPU.
    • For folder contents and details, see: microsam_cellpose3 and stardist.
    • More Pixi environments will be added soon.

    Load environment in VS Code

    Use the Pixi-created interpreter for scripts launched from VS Code:

    1) Open the Command Palette: Ctrl+Shift+P → "Python: Select Interpreter" → "Enter Interpreter Path" → "Browse your file system". 2) Navigate to the Pixi environment’s Python under the folder where you ran pixi install (Pixi shows the path via pixi info). 3) Select that interpreter. VS Code will now run using the Pixi environment.

    Make Pixi environment available to VS Code notebooks

    Register the environment as a Jupyter kernel so it appears in the notebook kernel selector:

    pixi run python -m ipykernel install --user --name=appose_napari_ai --display-name "Pixi (appose_napari_ai)"

    Notes: - If you see "No module named ipykernel": add it first with pixi add ipykernel, then rerun the command above. - If kernels or DLL paths don’t load correctly (e.g., CUDA/CUDNN), start VS Code from an activated Pixi shell so it inherits environment variables:

    pixi shell
